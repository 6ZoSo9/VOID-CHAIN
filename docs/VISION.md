# VISION — VOID & Obelisk

**VOID** is a decentralized AI computation and validation network.
**Obelisk Wallet** is the user’s interface (the conduit).
Rewards are **VoidStones ($VOID)** — crystallized fragments of useful work.

## Principles
- **Opt‑in micro‑validation** from wallets.
- **Provable contributions**: data availability sampling, relay quality, watchdogs, proving jobs.
- **Tiered roles**: Ghost (light), Wraith (medium), Titan (heavy).
- **Ethical UX**: explicit consent, resource caps, immediate off switch.
